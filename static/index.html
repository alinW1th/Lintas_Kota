👥 Users

/users
Lihat daftar semua pengguna terdaftar. Mendukung pagination.

⚡ Keunggulan

Dibangun dengan Praktik Terbaik

Setiap komponen dipilih untuk performa, keamanan, dan kemudahan pengembangan.

🔒

JWT Authentication

Sistem autentikasi berbasis token stateless menggunakan python-jose. Token di-expire otomatis dan mudah diintegrasikan ke frontend manapun.

🗄️

SQLAlchemy ORM

Model relasional dengan relasi cascade. Siap migrasi dari SQLite ke PostgreSQL atau MySQL hanya dengan ganti satu baris di .env.

✅

Validasi Pydantic v2

Semua request dan response divalidasi secara otomatis. Error dikembalikan dengan pesan yang jelas dan terstruktur.

🛡️

Role-Based Access

Tiga level akses: publik, user login, dan admin. Setiap endpoint dilindungi dependency injection yang bersih dan reusable.

🌐

CORS Configured

Cross-Origin Resource Sharing sudah dikonfigurasi via .env. Tinggal tambah origin frontend Anda.

📄

Auto Swagger Docs

Dokumentasi interaktif Swagger UI dan ReDoc di-generate otomatis oleh FastAPI. Setiap endpoint langsung bisa dicoba dari browser.

🛠️ Tech Stack

Teknologi yang Digunakan

Kombinasi library Python terpercaya untuk membangun API production-ready.
