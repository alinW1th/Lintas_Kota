Instalasi & Setup

Ikuti 5 langkah berikut untuk menjalankan LintasKota API di lingkungan lokal Anda dalam hitungan menit.

1
Clone Repository
Unduh source code dari GitHub ke komputer lokal Anda.
2
Buat Virtual Environment
Isolasi dependency Python agar tidak bentrok dengan proyek lain.
3
Install Dependencies
Pasang semua library yang dibutuhkan dari requirements.txt.
4
Konfigurasi .env
Salin template dan sesuaikan variabel database, JWT, dan CORS.
5
Jalankan Server
Aktifkan server development dengan hot-reload. API siap digunakan!
$ terminal
# Clone repository ke direktori lokal git clone https://github.com/username/LintasKota_API.git # Masuk ke direktori proyek cd LintasKota_API
$ terminal
# Buat virtual environment Python python3 -m venv venv # Aktifkan virtual environment # macOS / Linux: source venv/bin/activate # Windows: venv\Scripts\activate
$ terminal
# Install semua dependency pip install -r requirements.txt # Isi requirements.txt: fastapi uvicorn[standard] sqlalchemy pydantic bcrypt python-jose[cryptography] python-dotenv python-multipart aiofiles psycopg2-binary
.env
# Salin template environment cp .env.example .env # ── Konfigurasi DATABASE ────────────────── DATABASE_URL=sqlite:///./lintaskota.db # PostgreSQL: postgresql://user:pass@host:5432/db # ── Keamanan JWT ────────────────────────── SECRET_KEY=ganti_dengan_string_acak_panjang ALGORITHM=HS256 ACCESS_TOKEN_EXPIRE_MINUTES=30 # ── Aplikasi ────────────────────────────── APP_NAME=LintasKota API APP_VERSION=1.0.0 DEBUG=True # ── CORS (pisahkan dengan koma) ─────────── ALLOWED_ORIGINS=http://localhost:3000,http://localhost:5173 # Generate SECRET_KEY yang aman: python3 -c "import secrets; print(secrets.token_hex(32))"
$ terminal
# Jalankan server dengan hot-reload python3 -m uvicorn main:app --reload # Atau dengan port custom: python3 -m uvicorn main:app --reload --port 8080 # ── Server berjalan di: ─────────────────── http://localhost:8000 # Landing page http://localhost:8000/docs # Swagger UI http://localhost:8000/redoc # ReDoc # Database SQLite otomatis terbuat # di file lintaskota.db saat pertama run ✓
⚠️
Penting untuk Production: Ganti DATABASE_URL ke PostgreSQL (Supabase / Neon), isi SECRET_KEY dengan string acak ≥32 karakter, dan set DEBUG=False sebelum deploy ke Vercel.
Keunggulan

Fitur Utama API

Dirancang dengan prinsip clean code dan production-ready architecture dari hari pertama.

🔒

JWT Authentication

Autentikasi stateless berbasis token menggunakan python-jose. Token ber-expire otomatis, mudah diintegrasikan ke frontend manapun.

🛡️

Role-Based Access Control

Tiga level akses: Publik, User, dan Admin. Setiap endpoint dilindungi dependency injection yang clean dan reusable.

🗄️

SQLAlchemy ORM

Model relasional dengan cascade delete. Siap migrasi dari SQLite ke PostgreSQL hanya dengan mengganti satu baris di .env.

✅

Validasi Pydantic v2

Semua request & response divalidasi otomatis dengan Pydantic v2. Error dikembalikan dalam format JSON yang konsisten dan jelas.

🎫

Seat Availability Check

Saat booking, sistem secara otomatis menghitung kursi yang tersedia secara real-time dan menolak request jika melebihi kapasitas rute.

🌐

CORS & .env Config

Konfigurasi CORS & environment variables sepenuhnya via .env. Tidak ada hardcoded value — aman dan fleksibel untuk berbagai environment.

Arsitektur

Struktur Direktori Proyek

Semua logic terpusat di satu file main.py untuk kemudahan development dan deployment.

LintasKota_API/
 ├── main.py
 ├── .env
 ├── vercel.json
 ├── requirements.txt
 ├── package.json
 ├── LintasKota_Postman.json
 ├── .gitignore
 ├── lintaskota.db
 └── static/
     └── index.html
main.py

Entry point utama. Berisi seluruh konfigurasi FastAPI, model ORM (User, TravelRoute, Booking), skema Pydantic, utilitas JWT, dan semua 15+ endpoint API dalam satu file terstruktur.

.env

Menyimpan semua konfigurasi sensitif: DATABASE_URL, SECRET_KEY, ALGORITHM, ALLOWED_ORIGINS, dan pengaturan token expiry. Tidak di-commit ke Git.

vercel.json

Konfigurasi deployment ke Vercel. Menentukan routing "@vercel/python" untuk semua request diteruskan ke main.py.

requirements.txt

Dependency Python: fastapi, uvicorn, sqlalchemy, pydantic, bcrypt, python-jose, python-dotenv, python-multipart, aiofiles, psycopg2-binary.

LintasKota_Postman.json

Koleksi Postman siap import. Berisi seluruh request endpoint yang sudah dikonfigurasi dengan environment variable untuk testing lokal maupun production.

API Reference

Daftar Endpoint

15+ endpoint tersedia, terorganisir dalam 4 grup: Auth, Users, Routes, dan Bookings.

🔐 Authentication
POST /register Registrasi akun pengguna baru (name, email, password, role) Publik
POST /login Login & dapatkan JWT access token Publik
GET /me Lihat profil pengguna yang sedang terautentikasi Login
👥 Users
GET /users Lihat semua pengguna (dengan pagination) Admin
GET /users/{id} Detail pengguna berdasarkan ID Admin
PUT /users/{id} Update nama, telepon, atau role pengguna Login
DELETE /users/{id} Hapus pengguna & seluruh data terkait (cascade) Admin
🗺️ Travel Routes
GET /routes Lihat semua rute perjalanan yang tersedia Publik
GET /routes/{id} Detail rute berdasarkan ID Publik
POST /routes Tambah rute travel baru ke sistem Admin
PUT /routes/{id} Update detail rute (harga, jadwal, kapasitas) Admin
DELETE /routes/{id} Hapus rute — booking terkait ikut terhapus (cascade) Admin
🎫 Bookings
GET /bookings Admin: semua booking · User: hanya milik sendiri Login
GET /bookings/{id} Detail booking — hanya pemilik atau admin Login
POST /bookings Buat booking + cek ketersediaan kursi real-time Login
PUT /bookings/{id} Update jumlah kursi atau status (confirmed/cancelled) Login
DELETE /bookings/{id} Hapus/batalkan booking — hanya pemilik atau admin Login
